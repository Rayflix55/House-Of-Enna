# MODERN | Minimalist Fashion Store

![MODERN Storefront](https://picsum.photos/seed/fashion-hero/1200/600)

A premium, high-performance e-commerce storefront designed for the modern minimalist. Built with **React 19**, **Vite**, and **Tailwind CSS v4**, this application offers a seamless, high-end shopping experience with a focus on clean aesthetics and fluid interactions.

## ✨ Features

- 🌓 **Intelligent Theme Switching**: Seamless transition between Dark and Light modes with persistent state.
- 🛍️ **Complete Shopping Flow**: From product discovery to a streamlined checkout experience.
- 🔍 **Advanced Search**: Real-time product and category suggestions with trending search highlights.
- 📱 **Mobile-First Design**: Fully responsive architecture optimized for all device sizes.
- 🎨 **Dynamic Product Details**: Interactive size and color selection with high-resolution image galleries.
- 📝 **Fashion Journal**: Integrated blog and Q&A section for community engagement and SEO.
- 🔗 **Social Sharing**: One-click product sharing to Facebook, Twitter, and Pinterest.
- 🚀 **Performance Optimized**: Built on Vite for lightning-fast load times and HMR.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/modern-fashion-store.git
   ```

2. Navigate to the project directory:
   ```bash
   cd modern-fashion-store
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```
